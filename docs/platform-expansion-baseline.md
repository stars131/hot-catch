# 平台扩展基线

更新时间：2026-07-16

## 当前冻结范围

- 创作与发布平台仅包含 `xiaohongshu`、`douyin`。
- 内容类型仅包含 `xhs_graphic`、`douyin_video_script`。
- 创作入口、选题、发布素材校验、账号授权和复盘指标均以这两个平台为边界。
- 模型连接包含 DeepSeek、OpenAI/ChatGPT、Grok；Skill 支持设置页管理，并可在单次创作中多选。

## 当前硬编码边界

- Prisma 的 `Platform`、`ContentKind`、`InputType` 和凭证枚举决定持久化边界。
- 创作路由、客户端联合类型、内容输出 Schema、发布素材规则和平台标签仍直接判断小红书/抖音。
- 发布依赖 AiToEarn 的账号、授权、素材上传、状态同步和指标回收协议。
- 小红书信息抓取已经使用适配器链；热点聚合也有独立来源描述，可作为后续信息同步的参考结构。

## 下一阶段目标结构

国外平台接入前先建立统一的 `PlatformCapability` 注册表，每个平台声明：

- 平台标识、展示名称和支持的内容类型；
- 信息同步适配器、URL 识别器和标准化数据映射；
- 创作提示模板、可用 Skill 和结构化输出 Schema；
- 发布适配器、素材约束、授权方式与失败码映射；
- 指标字段、同步窗口和数据可用性说明；
- 所需凭证类型、健康检查和降级方式。

页面和服务只消费注册表与统一接口，不再新增平台名称分支。数据库枚举变更、历史数据迁移、真实凭证和真实发布验收必须按平台独立交付。

## 后续平台候选

YouTube、TikTok、Instagram、X、Reddit 的优先级和接入方式留到下一阶段确认。本次生产发布不新增平台枚举、页面入口、模拟账号或假发布结果。
